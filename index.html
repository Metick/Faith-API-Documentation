Getting Started

Please understand that the documentation is still a Work in progress, and will be changed and modified. The current API version is V0.1 

local menuInitialized;

if not menuInitialized then
MakeWindow("My Menu", ICON_FA_HOME, 250, 250) --the icon uses font-awesome icons.
MakeToggle("My Toggle", "My Menu")
MakeButton("My Button", "My Menu")

menuInitialized = true
end

if FetchToggle("My Toggle", "My Menu") then
--Do something when the toggle is on
end

if FetchButton("My Button", "My Menu") then
--Do something when the button has been pressed
--NOTE: When you click a button this will return true until you fetch once, once fetched it'll make it false until the user clicks the button again.
end

An Example on how to setup the GUI

Creating Graphical User Interface

<bool> MakeWindow(string windowName, string icon, float SizeX, float SizeY)

Returns Whether the window was created or if it failed

Notes :

  • You're Required to create a GUI in order to use our Existing GUI, If you don't use CreateGUI the menu wont draw.
  • If it fails it's because the window name already exists

Create Button

<bool> MakeButton(string Name, string windowName)

If true it worked, otherwise window doesn't exist.

Fetch a Button

<bool> FetchButton(string Name, string windowName)

True = toggled, False = untoggled

Create Toggle

<bool> MakeToggle(string Name, string windowName)

If true it worked, otherwise window doesn't exist.

Fetch a Toggle

<bool> FetchToggle(string Name, string windowName)

If true it worked, otherwise window doesn't exist.

Create Slider

<void> Faith.Slider(string Name, int Out, int min, int max)

Returns false/true whether the toggle has been activated or not

Keeping a module on the SameLine as the other one

<void> Faith.SameLine()

sets the module after this void to the same line as the module above SameLine.

Create Text

<void> Faith.Text(string Name)

Displays text
